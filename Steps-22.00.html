CREATING CHARTGPT INSIGHTS PROMPT 22

                                                                     

1.    Click on the empty canvas and drag a Matrix from the visualizations pane

2.    Add ChatGPT_Insights column to the Values field 

3.    Right click and select conditional formatting then click on Icons -->Format style =Rules, Apply to: Values and totals, What field should we base this on? = ChatGPT_Insights, Summarization= Count, Icon layout= Icon only 

If value >=0 and <= Max = Colaberyandchatgpt Icon

4.    In the Formatting Tab-->Visual--> Values: Font = Segoe UI, size = 20, Color = Black, Column headers: Font = Segoe UI, size = 12, Color = Black, Background: #A0D1FF.

5.    In the Formatting Tab-->General-->Properties-->Size: Height = 94, Width = 186, Position: Horizontal = 168, Vertical = 102.
