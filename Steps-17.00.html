CREATING CUSTOMER BY MONTH/YEAR CHART 17


                               


1.  Right click on the Lead by Month/Year chart, select copy, then click on copy visual.

2.  Press Control + V button on your computer to paste it. Place it underneath the Lead by Month/Year chart.

3. Change the X-axis field well to Customer_Date column from vw_Hubspot_Leads table and Column y-axis, to Customer column from vw_Hubspot_Leads table. Then drag the following measures into Line y-axis (ClickRateMarketing Cust MoM% and OpenRateMarketing Cus MoM%)

3. Drag all these measures into the Tooltips field ClickRateMarketing Cust MoM%, OpenRateMarketing Cus MoM% and Lead 2 Customer Measure.

4. In the Formatting Tab-->x-axis: On, Font = Arial, size = 12, Color = #000000. Title: Off. 

y-axis: On, Font = Arial, size = 10, Color = #000000. Title: Off. Secondary y-axis: On, Font = Segoe UI, size = 10, Color = #000000. Title: Off. 

Legend: On, Position = Top left, Text: Font = Segoe UI, size = 12, Color = #000000. Title: On.

Gridlines: Horizontal: On style = Dotted, Color = #C8C6C4, width = 1

Column: Color: #094780

Line Colors:  Open% = #A38600, Click% = #C00037, Shaded area: Off. 

Markers: Off

Data labels: Off. 

5. In the Formatting Tab-->General-->Properties-->Size: Height = 313, Width = 537, Position: Horizontal = 891, Vertical = 410, Title: Customers by Month/Year, Font = Arial, Size = 15, Color = #252423, Background: #A0D1FF, Center, Effect: Background: On, Color = White, Visual Border: #CCCCCC

Header icons: On: Icon: #333333

Tooltips: On, Font = Segoe UI, Size = 10, Label Color = White, Value Color = White and Background: Black
