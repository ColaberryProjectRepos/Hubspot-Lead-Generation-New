CREATING DATE REFRESH 14

                                                                         

1. Go to power query, right click in the query panel to the left
2. Select New Query in the submenu
3. In the next submenu, select Blank Query    
4. In the query bar type:
    =DateTime.LocalNow
5. Hit Enter and rename the query to Date refreshed.
