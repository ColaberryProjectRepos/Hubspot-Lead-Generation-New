CREATING CUSTOMER DRILL THROUGH MATRIX 20

      


1.    Duplicate the Overview Page and rename it Customer Drill-Through

2.    Delete Customers Source Drill down, Lead by Month/Year  and Customer by Month/Year chart from the page 

3.    Click on the empty canvas and drag a Matrix from the visualizations pane

4.    Drag Total, Customer_Date and Actual Lead Source to the Rows from vw_Hubspot_Leads Table

5.    Add the following columns and measures one by one to the Values field


a)    Avg Click Rate - from Key measures

b)   ClickRateMarketing Cust MoM% - from Key measures

c)    Average OpenRate - from Key measures

d)   OpenRateMarketing Cus MoM% - from Key measures  

e)   Customers – from vw_Hubspot_Leads Table

f)    PercentEnrolled – from vw_Hubspot_Leads Table

6. Drill through field well, drag lead_date 

7.    In the Formatting Tab-->Visual--> Values: Font = Segoe UI, size = 13 Column headers: Font = Arial, size = 13, Color = Black, Background: #A0D1FF.

Column subtotals: On, Column: Values: Font = Segoe UI, size = 13, Text Color = Black

Row subtotals= On, Per row level = On, Row level = Total, Rows: Show Subtotal =Off

Column grand total: Values: Font = Segoe UI, size = 13, Text Color = Black

URL icon: Values = On       

8.    In the Formatting Tab-->General-->Properties-->Size: Height = 529, Width = 1258, Position: Horizontal = 171, Vertical = 195.
