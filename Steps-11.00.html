CREATING MEASURES 11

                                                               


First we create a measure table where all our measures will be stored by;
1. Clicking on the Enter data icon on the top of the report page
2. Name the table
3. Click on Load

Now that you have created the measure table, right click on it from the data pane on the right side of the report page and click new measure to add a measure.
