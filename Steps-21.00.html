CREATING CHARTGPT ICON 21

                                                                                

1.       Get the icon image from zip folder.

2.       Paste the follow json query on a note pad

 {

           "name": "",

 
          "icons": [{

 
                   "url": ""
     

         }]

 }
   
3.       Open your web browser to https://www.base64-image.de/


4.       Drag and drop the image on the drag and drop images space on the site.

5.       Click on show code, then on For use in <img> elements: select copy to clipboard.

6.       Paste the content of the clipboard copied in between the “” in front of "url": "xxx" on the notepad.

7.       Give a meaningful name to your icon in between the “” in front of "name": "xyz".

8.       Save the notepad as .json, select all files and save.

9.       Open up Power bi desktop, click on view, then click on the drop down beside the theme, then select browse theme, upload your save .json file.
