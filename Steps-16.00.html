CREATING LEAD BY MONTH/YEAR CHART 16

                           


1. Click on an empty space on the canvas, and from the visualizations pane select Line and Clustered Column Chart.

2. Drag Lead_Date column from vw_Hubspot_Leads table to the X-axis. Follow by Prospect column onto Column y-axis, then drag the following measures into Line y-axis (ClickRateMarketing Lead MoM%, OpenRateMarketing Lead MoM%, and Enrollment Rate)

3. Drag all these measures into the Tooltips field: ClickRateMarketing Lead MoM%, OpenRateMarketing Lead MoM%, and Enrollment Rate

4. In the Formatting Tab-->x-axis: On, Font = Arial, size = 12, Color = #000000. Title: Off. 

y-axis: On, Font = Arial, size = 10, Color = #000000. Title: Off. Secondary y-axis: On, Font = Segoe UI, size = 10, Color = #000000. Title: Off. 

Legend: On, Position = Top left, Text: Font = Segoe UI, size = 12, Color = #000000. Title: On.

Gridlines: Horizontal: On style = Dotted, Color = #C8C6C4, width = 1

Column: Color: click on fx, Format: Gradient, What field should we based this on: Lead 2 Customer Measure, Minimum: #FFFFFF and Maximum: #717BC5

Line Colors:  Enrollment Rate% = #6B007B, Open% = #A38600, Click% = #C00037, Shaded area: Off. 

Markers: Off

Data labels: Off. 

5. In the Formatting Tab-->General-->Properties-->Size: Height = 308, Width = 537, Position: Horizontal = 891, Vertical = 102, Title: Lead by Month/Year, Font = Arial, Size = 15, Color = #252423, Background: #A0D1FF, Center, Effect: Background: On, Color = White, Visual Border: #CCCCCC

Header icons: On: Icon: #333333

Tooltips: On, Font = Segoe UI, Size = 10, Label Color = White, Value Color = White and Background: Black
